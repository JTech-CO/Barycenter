## Just completed

- Added inertial, massive-only barycentric, and primary-pair rotating-frame position/velocity transforms with 3D round-trip gates.
- Added reversible physical AU·yr·M☉ ↔ canonical CR3BP length, time, and velocity normalization.
- Implemented canonical effective potential, analytic gradient, equations of motion, Jacobi constant, and finite zero-velocity grid sampling.
- Added bracketed L1–L3 solves with retained failure context and analytic L4/L5 coordinates.
- Added equilibrium and tadpole massless-tracer presets integrated through the existing Yoshida4 N-body path.
- Passed 11 analysis tests: L1–L5 residual max `7.40e-14`; 20-period tadpole Jacobi max error `1.61e-10`.
- Generated byte-stable JSON/Markdown Analysis Alpha evidence; the full fast suite now has 61 passing tests.
